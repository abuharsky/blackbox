# Pub.dev scoring hints (RU)

Этот файл — краткая памятка, что обычно улучшает pana/pub score:

- `repository:` в pubspec.yaml (и реально существующий URL)
- `issue_tracker:` (опционально)
- качественный `description`
- `topics:` (до 5) — если релевантно
- `dart analyze` без warning/error
- README с примерами и ссылками на docs
- CHANGELOG с актуальной версией
- LICENSE
- `example/` (опционально, но помогает восприятию)

Для публикации:
- `dart pub publish --dry-run`
- `dart pub publish`
